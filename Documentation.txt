Project Summary:
The Movie Recommendation System is a Python based application
that simplifies the process of selecting movies for individuals
or groups. By leveraging user preferences and favorite genres
the systems gives personalized movie recommendations that make 
finding a movie more enjoyable for the user. Users can enter
there preferences manually, explore the genres, or take a quiz
all the recive suggustions. The program runs offline, is simple
to use doesn't require to create an account or login, and 
doesn't require internet access. 


How to run program:
To run the project, ensure you have the following installed:

Python 3.7 or higher. No third-party libraries are required. This project runs using only Python's standard library.
INSTALLATION & USAGE

Clone the Repository

Go to the Github Repo Page & Click the green "Code" button, then copy the HTTPS URL
Open Terminal or Command Prompt

Windows: Open "Command Prompt"
Mac/Linux: Open "Terminal"
Run "git clone" with the Repo URL

(This will create a new folder with project files)
Navigate Into the Project Folder

Ex. "cd movie-recommendation-system"
Now that you're inside the folder you can run the python program
Run the Program.

Ex. "python main.py" but if your system uses python3 instead use "python3 main.py"


How to use Program:
Once launched:

Asks how many movie recommendations you want

Offers multiple options to choose genres:

Type genres directly (e.g., Action, Comedy)

Type list to see all available genres

Type quiz to take a short mood-based quiz

Type random to get surprise recommendations

Displays your recommended movies

Asks if you’d like to run it again
